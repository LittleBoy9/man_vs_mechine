About / Project note

Why this website exists

It is a simple multi-page HTML site that behaves like a small publication. The theme is conflict, but the real subject is collaboration: what happens when people and software share authorship.

Site structure

What is inside

  • Shared navigation across all pages.
  • One reusable stylesheet and one tiny script.
  • Local JPG and WEBP poster assets with readable text.
  • Remote poster-style images that also contain text.
Naming note

Why “Mechine” stays

The spelling is preserved because it gives the project a rough, authored identity. It feels less like a startup landing page and more like a homemade editorial artifact.

Production note

Asset approach

The site now mixes your local JPG and WEBP assets with the custom SVG set, so the visual system feels less synthetic and more varied.

I can also extend this into more pages such as article detail pages, an authors page, or a contact page without changing the structure much.

Static and easy to expand

Everything here is plain HTML/CSS/JS, so the site can be hosted anywhere or expanded page by page.
